# OpenAI MCP Server Guide

## Configuration
The OpenAI MCP server is configured in `.kiro/settings/mcp.json`.

## Setup Steps
1. Ensure you have `uv` and `uvx` installed
2. Add your OpenAI API key to the environment variable `OPENAI_API_KEY`
3. The server will automatically connect when Kiro starts

## Usage
Once configured, you can use OpenAI capabilities through MCP tools in your Kiro workspace.

## API Key Setup
Replace `your-api-key-here` in the mcp.json file with your actual OpenAI API key.
You can get an API key from: https://platform.openai.com/api-keys
